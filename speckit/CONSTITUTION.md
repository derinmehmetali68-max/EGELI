# Speckit Constitution — Okul Kütüphane Otomasyonu
- Para/ceza hesaplaması YOK.
- Mimari: Express + SQLite + React + Tailwind.
- Güvenlik: JWT, Helmet, RateLimit. Üretimde HTTPS zorunlu.
- Veri bütünlüğü: ISBN normalizasyonu, toplu import doğrulama.
- UI: Lise seviyesine uygun sade dil.
- Raporlama: PDF (gecikenler), CSV/XLSX export. Ücret alanları yok.
